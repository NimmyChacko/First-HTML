Introduction to HTML

The HyperText Markup Language or HTML is the standard mark up language that web browsers use to interpret and compose text, images, and other material into visible or audible web pages.It is used to give structure to a web page.

Mark up

HTML markup consists of several key components, including those called tags (and their attributes), character-based data types, character references and entity references.The first tag in such a pair is the start tag, and the second is the end tag (they are also called opening tags and closing tags).

Some examples of tags are:
  • h1 to h6 – headings(large font to small)
  • p – paragraph
  • br – break
  • em – emphasis(italics)
  • ol – ordered list
  • ul – unordered list
  • strong or b(old version) – bold
  • img – image

Attributes

HTML attributes are a modifier of a HTML element type. Some examples are:

  1. scr – source
  2. width
  3. height
  4. alt – alternate

Syntax

Syntax is the arrangement of elements and attributes to create well-formed documents.(like correct grammar in a language).
Semantic HTML, also known as semantic markup, refers to the use of HTML tags that convey the meaning—or semantics—of the content contained within them.
